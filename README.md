# Coronary Artery Disease ML Analysis
A machine learning project applying supervised and unsupervised learning techniques on coronary artery disease (CAD) dataset.

This project includes classification modeling for disease prediction and K-Means clustering for patient segmentation analysis.

# Project Objective
To analyze coronary artery disease data by:
- Building classification models to predict CAD presence
- Evaluating model performance using confusion matrix
- Implementing K-Means clustering for exploratory patient grouping
- Applying Elbow method and SSE evaluation for optimal cluster selection

# Project Components
## Exploratory Data Analysis
  - Data inspection & cleaning
  - Correlation analysis
  - Outlier detection
## Supervised Learning (Classification)
  - Data normalization
  - Train-test split
  - Multiple classification models
  - Voting Classifier
  - Confusion Matrix evaluation
  Target variable: Cath
## Unsupervised Learning (Clustering)
  - Manual implementation of K-Means algorithm
  - Centroid initialization
  - Iterative centroid update
  - SSE calculation
  - Elbow Method
  - Clustering using:
  - 1 feature (Age)
  - 2 features (Age & BMI)

# Skills Demonstrated
- Classification Modeling
- Ensemble Learning (Voting Classifier)
- K-Means Algorithm Implementation
- Model Evaluation (Confusion Matrix)
- Feature Scaling
- Data Exploration & Cleaning
